# SnowDeformation
该项目是做了一个雪地制作脚印的教程！

![unreal-engine-snow-12](https://user-images.githubusercontent.com/26760292/143068568-952d31b5-db56-4e71-8967-b638cfc3ce32.gif)


课程步骤：

        该项目的制作过程大概分以下几个制作过程：

        1.开始准备

        2.雪道实施

        3.从底部捕获

        4.创建深度检查材质

        5.创建场景捕获

        6.设置捕获区域大小

        7.存储捕获大小

        8.变形景观

        9.投影呈现器目标

        10.使用呈现器目标

        11.创建持久跟踪

        12.创建持久缓冲区

        13.写回捕获

        14.移动捕获

        15.以离散步骤移动捕获

        16.移动持久性缓冲区

        17.从这里去哪里？
